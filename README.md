# AI-Powered Telecom Training Assistant

## Project Overview
This project is an AI-powered educational assistant developed using IBM watsonx.ai, IBM Granite Models, LangFlow, and Retrieval-Augmented Generation (RAG). It helps users learn telecommunication concepts by answering questions from uploaded PDF documents.

## Features
- Answers telecom-related questions
- Explains complex concepts
- Summarizes technical documents
- Generates quizzes
- Uses RAG for accurate document-based responses

## Technologies Used
- IBM watsonx.ai
- IBM Granite Models
- LangFlow
- IBM Cloud
- FAISS/ChromaDB
- Retrieval-Augmented Generation (RAG)
